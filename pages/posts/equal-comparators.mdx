No último post ([veja aqui](https://bit.ly/3xZabw9)) utilizei duas comparações (**“igual a” (==)** e **“idêntico a” (===)**) que talvez não sejam claras para todos.

Se usarmos o operador **==** em uma condição:
 - Ela só será verdadeira se o valor da esquerda for o mesmo valor da direita.
 - Compara por 'resultado'.
 - Converte o que você quer comparar e verifica.

Se usarmos símbolo de “idêntico a” **===**:
 - Não só irá comparar os valores dos dois lados da equação, como também verificará se eles são do mesmo tipo.

Deixei alguns exemplos na imagem.

```
  // using ==
  null == undefined 	// true
  true == “true”  	// true
  5 == “5”        	// true
  true == true    	// true
  5 == 5          	// true
    
  // using ===
  null === undefined	// false
  true === “true” 	// false
  5 === "5"         	// false
  true === true   	// true
  5 === 5       		// true
```
