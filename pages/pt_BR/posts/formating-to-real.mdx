Já precisou formatar o resultado de uma conta para uma moeda, como por exemplo, o Real?

No JavaScript temos um método chamado toLocaleString() que converte um número para uma string, já tratando a questão do arredondamento e convertendo para a moeda do país que queremos, no nosso caso, o Real, tornando a tarefa do programador muito mais simples.

Está ai mais uma dica simples e poderosa.

Referências:

[Mozilla](https://mzl.la/3xYeGXM)

[Alura](https://bit.ly/3zKlYiU)


EXEMPLO:

```
const value = 20.555;

const valueFormatedToBRL = 
      value.toLocaleString(
		'pt-BR', 
    	{ 
  	  	style: 'currency', 
      	currency: 'BRL' 
    	}
	);

console.log(valueFormatedToBRL); // R$ 20,56
```
